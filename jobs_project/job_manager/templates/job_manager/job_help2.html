Help

Upload Files: form to enter the nlogo/account parameters to run an experiment in SGE Cluster.

Output Files: shows the execution results on SGE Cluster (by job_id).

Login/Logout: authentication in order to use Hpcnetlogo.


Job List:

Number: local sequence id.

JobId: experiment id.

User: user in the SGE Cluster.

Cluster: name.domain or IP of SGE Cluster.

Date: Upload Time.

State: upload experiment but not submited, submited work (click again to submit new instances of the same work)

Delete: remove the experiment & parameter.
